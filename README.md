# The Asylum

A personal website built for [Neocities](https://neocities.org) — a digital sanctuary where technology serves humanity.

**Live site:** [jeffkazzee.neocities.org](https://jeffkazzee.neocities.org)

## About

The Asylum is a personal website with a "Whimsygothic Cyber Ghost" aesthetic — Victorian paranormal meets retro computing. Built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools, no npm.

## Features

- **Gate Animation** — Theatrical entrance with wrought iron gates that swing open
- **Resource Library** — 40+ curated tools for developers and designers
- **Custom SVG Icons** — Hand-crafted Victorian-style ornate icons
- **Responsive Design** — Works on all devices
- **Accessibility** — Skip links, keyboard navigation, reduced motion support

## Tech Stack

- HTML5
- CSS3 (Custom Properties)
- JavaScript + [Alpine.js](https://alpinejs.dev/)
- Hosted on [Neocities](https://neocities.org)
